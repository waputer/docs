Hello World in PL3
==================


## QuickStart

Use the following steps to build the application and run it in the Waputer
terminal:

* Downlaod the PL3 compiler via `wget https://github.com/waputer/pl3/releases/download/v1.0.0-dev9/pl3-1.0.0-dev9-bin.tar.xz`
* Get `hello.pl3` or write your own.
* Complie using `pl3-1.0.0-dev9-bin/pl3 hello.pl3 -t wapr -i pl3-1.0.0-dev9-bin/rt.wapr.pl3mod -o hello.wapr --dynlib rt`
* Open the developer terminal at [waputer.dev](https://waputer.dev).
* In the developer terminal, upload the program by running `upload /prog`.
* Still in the terminal, run the program with the command `/prog`.
