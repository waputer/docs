Hello World in C
================

The hello world example is an extremely bare metal example of C code written
to run on the Waputer. This is not intended to be the primary method for
creating Waputer programs. It demonstrates how some of the core pieces of
Waputer work, and how more sophisticated tools can be written on top. Think of
it like a hello world example written in assembly.


## QuickStart

Use the following steps to build the application and run it in the Waputer
terminal:

* Install the programs `clang` and `wasm-ld`. 
* Build the program by running `make`.
* Open the developer terminal at [waputer.dev](https://waputer.dev).
* In the developer terminal, upload the program by running `upload /prog`.
* Still in the terminal, run the program with the command `/prog`.
